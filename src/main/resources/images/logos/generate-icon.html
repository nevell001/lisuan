🖼️ 狸算(LiSuan)收银系统图标生成器

📝 使用说明

  1. 下方显示了自动生成的应用图标预览
  2. 点击 "下载 PNG 图标" 按钮保存为 app-icon.png
  3. 将下载的文件保存到项目目录：
    src/main/resources/images/logos/app-icon.png
  4. 重新编译项目：mvn clean package
图标尺寸: 256 x 256 像素

💡 图标说明

设计元素：

  • 背景色： #4A82BA（项目主色调蓝色）
  • 图标元素： 收银机/计算器 + 购物车
  • 文字： "狸算" 或 "¥" 符号
  • 风格： 现代扁平化设计，圆角边框

如需自定义图标：

  • 访问 favicon.cc 在线编辑
  • 使用 Figma、Sketch 或 Photoshop 设计
  • 从 Flaticon 下载免费图标
